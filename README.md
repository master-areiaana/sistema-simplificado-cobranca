# Sistema de Cobrança (Unificado — GitHub + Supabase)

Versão única do sistema, sem Base44. Todo o passo a passo de instalação,
configuração do Supabase, deploy e testes está em **[LEIA-ME.md](./LEIA-ME.md)**.

Início rápido:
```bash
npm install
cp .env.example .env   # preencha com as credenciais do Supabase
npm run dev
```
Banco: rode `supabase-schema.sql` uma vez no SQL Editor do Supabase.
